#Эко-калькулятор

Простая программа на Python, которая рассчитывает твой углеродный след по трём показателям:
- 💡 Электричество
- 🚗 Поездки на машине
- 🗑️ Бытовой мусор

## Тип проекта
> Консольный калькулятор углеродного следа на Python

## Библиотеки
Никаких дополнительных библиотек не нужно — только стандартный Python.

## Как запустить
1. Установи Python (если ещё не установлен) с [официального сайта](https://www.python.org/downloads/)
2. Скачай файл `co2.py`
3. Открой терминал в папке с файлом
4. Напиши `python co2.py`

## Коэффициенты расчёта (упрощённые)
| Что считаем | Формула |
|-------------|---------|
| Электричество | 1 кВт·ч = 0.4 кг CO₂ |
| Машина | 1 км = 0.15 кг CO₂ |
| Мусор | 1 кг = 0.5 кг CO₂ |

## Что умеет программа
- Задаёт 3 простых вопроса
- Переводит всё в годовой выброс CO₂
- Даёт совет: хорошо / нормально / много
