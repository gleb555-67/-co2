# Эко-калькулятор
Программа на Python, которая рассчитывает твой углеродный след по трём показателям:
- 💡 Электричество
- 🚗 Поездки на машине
- 🗑️ Бытовой мусор
- Расчет co2 по показанию счетчика

## Тип проекта
> Продвинутый калькулятор углеродного следа на Python

## Библиотеки
Скоро будут

## Коэффициенты расчёта (упрощённые)
| Что считаем | Формула |
|-------------|---------|
| Электричество | 1 кВт·ч = 0.4 кг CO₂ |
| Машина | 1 км = 0.15 кг CO₂ |
| Мусор | 1 кг = 0.5 кг CO₂ |
| Показания счётчика (электричество) | 1 кВт·ч = 0.4 кг CO₂ |

## Что умеет программа
- Спрашивает пользователя 3 простых вопроса
- По возможности пользователь может отправить показания электричества
- Переводит всё в годовой выброс CO₂
