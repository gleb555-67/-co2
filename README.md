# 🌱 EcoCO2 Bot — Telegram-бот для оценки углеродного следа

**Автор:** Глеб Кондаков 
**Стек:** Python, pyTelegramBotAPI, SQLite

Бот помогает быстро и просто оценить личный углеродный след по трём основным категориям:  
электричество, автомобиль и бытовые отходы.  
Создан в рамках изучения экологических проблем

## Коэффициенты расчёта (упрощённые)
| Что считаем | Формула |
|-------------|---------|
| Электричество | 1 кВт·ч = 0.4 кг CO₂ |
| Машина | 1 км = 0.15 кг CO₂ |
| Мусор | 1 кг = 0.5 кг CO₂ |
| Показания счётчика (электричество) | 1 кВт·ч = 0.4 кг CO₂ |

## 🎯 Цель проекта

- **Повысить осведомлённость** о влиянии повседневных привычек на климат.
- **Показать простой способ** количественно оценить свой вклад в выбросы CO₂.
- **Предоставить рекомендации** по снижению углеродного следа (ссылки на материалы ООН).

Бот будет полезен всем, кто интересуется экологией, хочет разобраться в своём воздействии на природу или просто попробовать работу с Telegram Bot API.

## ⚙️ Функции и особенности

- **Пошаговый диалог**  
  Бот последовательно запрашивает:
  - расход электроэнергии (кВт·ч/мес.),
  - пробег автомобиля (км/мес.),
  - количество мусора (кг/мес.).

- **Расчёт годового следа**  
  На основе усреднённых коэффициентов пересчитывает в кг CO₂ в год и выводит сводную таблицу.

- **Сохранение истории**  
  Все расчёты записываются в локальную базу данных SQLite (`co2.db`).  
  Можно анализировать свою динамику или статистику пользователей.

- **Команда `/cancel`**  
  Позволяет прервать диалог на любом шаге и начать заново.

- **Поддержка фото** (заготовка)  
  Бот вежливо сообщает, что пока не умеет распознавать показания счётчиков, но готов принять число.

- **Полезные ссылки**  
  После каждого расчёта предлагается ссылка на официальные советы ООН по снижению углеродного следа.

- **Обход блокировок (опционально)**  
  Бот умеет работать через SOCKS5-прокси (например, Tor) для доступа к API Telegram в странах с ограничениями.
